动态域名解析简介

名称 解释
中文名 动态域名服务
外文名 Dynamic Domain Name Server
缩写 DDNS
作用 实现动态域名解析
原理 将用户的动态IP地址映射到一个固定的域名解析服务上
服务器模式 客户端/服务器

适用环境

名称 条件
域名服务商 DNSPod
路由器 爱快软路由
IP 动态公网

开始

  • 本教程域名以 bt.ddos-sb.cc 为例

添加解析

在解析处随便添加一个 ipv4 地址

无需完全同步 106.46.46.104 为方便使用 1.1.1.1 即可

获取 DNSPod Token

  • 在 DNSPod 依次点击 头像 我的账号

  • 在 账号中心 → API密钥 → DNSPod token 中 创建密钥

配置 动态域名

  • 在爱快路由后台中找到 高级应用 → 动态域名 并填好相关信息

红框内的配置要与我相同喔

  • 保存 启用 完成

到这里就做好 DDNS 了

DMZ主机

  • 这里简单提一下 DMZ
  • 有 公网IP 时 若 DMZ 主机 IP 为 192.168.1.3
  • 访问 公网IP:端口 等同于访问 192.168.1.3:端口

DMZ主机 等于直接将 DMZ主机 暴露
因此
我们在配置一个拥有 DMZ区 的网络的时候通常定义以下的 访问控制策略 以实现DMZ区的 屏障 功能。

  • DMZ 允许外网直接访问
  • 内网可以访问外网
  • 内网可以访问 DMZ
  • 外网不能访问内网
  • DMZ 访问内网有限制
